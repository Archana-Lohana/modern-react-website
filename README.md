## Prospective Customer Segmentation Dashboard

**Live Demo:** [View the Dashboard](https://archana-lohana.github.io/modern-react-website/)


A modern, interactive React application that visualizes and categorizes potential banking customers based on satisfaction and access to financial products. This dashboard is designed to help financial institutions understand customer segments and make data-driven decisions.

## Features

- **Dynamic Customer Cards:** Display customer segments with relevant details, including images, tags, and brief introductions.
- **Interactive UI:** Smooth hover effects, buttons, and scroll-free layouts for a clean user experience.
- **Segment Classification:** Categorizes customers into five key groups:
  - Satisfied – Prime customers satisfied with current products.
  - Underserved – Prime customers not fully satisfied.
  - Underbanked – Limited access to traditional credit.
  - Unbanked – No formal banking relationship.
  - Credit Invisible – Little or no credit history.
- **Responsive Design:** TailwindCSS ensures a mobile-first and visually appealing layout.

## Technologies Used

- React.js – Component-based UI for modular and maintainable code.
- Tailwind CSS – Rapid styling and responsive design.
- JavaScript ES6+ – Modern JavaScript features for clean and efficient code.

 to view the app.

## Usage

->Browse customer segments visually.

->Click segment buttons to view tags and categories.

->Use this dashboard to identify target audiences for banking services.

## Customization

->Add new customer segments by extending the users array in App.js.

->Customize colors, tags, and images directly in the object properties.

->Modify styling through Tailwind CSS classes in each component.
